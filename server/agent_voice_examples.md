## How It Works

### Step 1: Agent Sets Profile
```
Name: Sarah Chen
Agency: Prestige Property
Tone: Luxury
Catch Phrases: ["A rare opportunity"]
```

### Step 2: AI Learns From Past Listings
- Analyzes sentence structure
- Identifies favorite words
- Detects formality level
- Extracts signature phrases

### Step 3: Property Type Detection
```
$4.2M property → Luxury tone
Family home → Warm tone
Investment unit → Professional tone
```

### Step 4: Personalized Generation
- Combines agent profile + property type
- Generates unique content
- Sounds like the agent wrote it
