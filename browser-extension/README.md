## 🚀 Key Features

### 1. Property Scraper
**On realestate.com.au / Domain:**
```
1. Agent views property listing
2. Clippy icon appears on page
3. Click "Add to Clippy"
4. Auto-extracts:
   - Address, price, beds, baths
   - Photos, description
   - Agent contact info
5. Creates Clippy listing instantly
```

**Time Saved:** 5 minutes per listing

### 2. Quick Lead Capture
**Anywhere on web:**
```
1. Select text (name, email, phone)
2. Right-click → "Add to Clippy"
3. Choose: New Lead / Add to Existing / Ignore
4. Done!
```

**Use Case:** 
- Agent reading Facebook comments
- Sees "Hi, I'm interested in 123 Main St"
- Right-click → Save as lead → Done

### 3. AI Reply Generator
**In Facebook Messenger:**
```
1. Prospect messages on Facebook
2. Clippy shows "AI Reply" button
3. Click → Generates personalized reply
4. Edit if needed → Send
```

**Time Saved:** 2 minutes per reply

### 4. Sidebar Mode
**Always available:**
- Press `Ctrl+Shift+C` → Opens Clippy sidebar
- Access leads, tasks, AI assistant
- Never leave current page
- Works on any website

### 5. Email Parser
**In Gmail/Outlook:**
```
1. Open inquiry email
2. Clippy scans automatically
3. "Create Lead" button appears
4. Click → Lead created with all info
5. Auto-generates reply draft
```

**Time Saved:** 3 minutes per email
